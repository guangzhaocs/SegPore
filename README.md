## SegPore Workflow


### Environment setup
```
conda env create -f environment.yml
conda activate segpore_env
```

### Download SegPore source code and demo data
```
git clone 
```

### Step 1: Basecalling, mapping and preprocessing
```
sh basecalling.sh
```

sh todo.sh
